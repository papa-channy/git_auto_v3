요청하신 `kakao.py` 스크립트의 주요 기능과 로직을 300 tokens 내외로 요약해 드리겠습니다.

### 주요 기능
1. **카카오톡 메시지 전송 기능**: 이 스크립트는 카카오톡을 통해 메시지를 전송하는 기능을 제공합니다.
2. **토큰 관리**: 카카오 API를 사용하기 위한 토큰을 로드, 저장, 갱신하는 기능을 포함합니다.

### 주요 로직
- **`load_tokens` 함수**: 저장된 카카오 API 토큰을 로드합니다. `TOKEN_PATH`에 저장된 JSON FILE을 읽어 토큰 정보를 가져옵니다.
- **`save_tokens` 함수**: 토큰 정보를 저장합니다. 입력받은 데이터를 JSON 형태로 `TOKEN_PATH`에 저장합니다.
- **`refresh_access_token` 함수**: 리프레시 토큰을 사용해 새로운 액세스 토큰을 발급받습니다. 카카오 API 서버에 요청을 보내 새로운 토큰을 받아옵니다.
- **`build_headers` 함수**: API 요청 헤더를 구성합니다. 액세스 토큰을 포함하여 인증 정보를 헤더에 추가합니다.
- **`_send_msg` 함수**: 카카오톡 메시지를 전송합니다. 구성한 헤더와 메시지를 카카오 API에 전송하고, 응답 결과를 확인합니다.
- **`send` 함수**: 커밋 메시지와 상태를 받아 카카오톡으로 메시지를 전송합니다. 필요 시 토큰을 갱신하여 사용합니다.
- **`ping` 함수**: 카카오톡 연결 상태를 테스트합니다. "✅ [Ping 테스트] 카카오 알림 연결 성공" 메시지를 전송하여 연결을 확인합니다.

### 레포 전체 구조에서의 역할과 연계성
이 스크립트는 `notify` 디렉토리에 위치하여, 프로젝트 내에서 발생하는 특정 이벤트(예: 커밋 메시지 전송 등)에 대해 카카오톡을 통해 알림을 보내는 역할을 합니다. 다른 스크립트(예: `scripts/gen_msg.py`, `scripts/llm_router.py` 등)와 연계하여 사용될 수 있으며, 전체 프로젝트의 자동화 및 알림 체계의 일부로 기능합니다. 

특히, `runall.py`나 다른 스크립트에서 이 모듈을 임포트하여 카카오톡 알림 기능을 사용할 수 있습니다. 또한, `config/kakao.json`과 같은 설정 FILE에 저장된 카카오 API 관련 설정을 읽어와서 사용합니다.
