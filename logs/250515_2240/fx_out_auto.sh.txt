## 📝 스크립트 요약: auto.sh

### 주요 기능
`auto.sh` 스크립트는 VSCode가 종료될 때 자동으로 실행되도록 설정된 스크립트입니다. 주요 기능은 다음과 같습니다:
1. **프로젝트 자동화 실행**: 특정 조건에 따라 프로젝트 내 다양한 Python 스크립트를 실행합니다.
2. **로그 및 결과 관리**: 실행 로그를 저장하고 결과를 관리합니다.
3. **환경 설정**: 필요한 환경 변수를 설정하고, 특정 디렉토리로 이동하여 작업을 수행합니다.

### 로직
1. **환경 설정**: 
   - 실행 경로 설정 및 필요한 디렉토리로 이동
   - 환경 변수 설정 (예: `PYTHONPATH`)

2. **스크립트 실행**:
   - `runall.py` 스크립트를 실행하여 다른 Python 스크립트들을 순차적으로 실행합니다.

3. **로그 관리**:
   - 실행 로그를 특정 디렉토리(`logs/`)에 저장합니다.

### 레포 전체 구조에서의 역할
`auto.sh`는 프로젝트의 자동화 실행을 담당하며, `runall.py`와 연계되어 다양한 Python 스크립트를 실행합니다. 주요 Python 스크립트들은 `scripts/` 디렉토리에 있으며, 이들은 데이터 처리, LLM(대형 언어 MODEL NAME) 실행, 알림 전송 등 다양한 기능을 수행합니다.

### 연계성
- **`runall.py`**: `auto.sh`에서 호출되어 순차적으로 다른 Python 스크립트를 실행합니다.
- **`scripts/` 디렉토리 내 스크립트들**: 데이터 처리, LLM 실행, 알림 전송 등의 기능을 수행합니다.
- **`config/` 디렉토리 내 설정 FILE들**: 실행에 필요한 설정 값들을 제공합니다.
- **`logs/` 및 `results/` 디렉토리**: 실행 로그 및 결과를 저장합니다.

### README와의 관련성
README에 설명된 대로, `auto.sh`는 VSCode 종료 시 자동으로 실행되도록 Windows 작업 스케줄러에 등록됩니다. 이를 통해 사용자가 직접 스크립트를 실행하지 않아도 자동화가 이루어집니다.
